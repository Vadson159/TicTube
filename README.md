## About

TicTube is a YouTube client designed specifically for Android Wear OS devices. It is built using modern Android development practices to deliver a smooth and native experience right on your wrist. 

**Key Features & Technologies:**
* **Platform:** Android (Wear OS)
* **Language:** Kotlin
* **UI:** Jetpack Compose for Wear OS (using `ScalingLazyColumn`, `Chip`, etc.)
* **Video Player:** Media3 (ExoPlayer) with full support for Wear OS Media Notifications.
* **Gestures:** Custom player controls including single tap for UI toggle, double tap for seeking (10 seconds), and pinch-to-zoom/pan functionality.
* **Authentication:** Google Sign-In API integration for a personalized experience.
* **Backend Integration:** Uses the reliable NewPipe Extractor for fetching YouTube data (search, channels), and Google API for subscriptions.
* **Content:** Support for the main feed, Shorts feed, history with resume timing, and subscriptions with channel avatars.
